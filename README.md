## Projects

### Music Virality Prediction
*What makes a song blow up before it does?*

Collects real-time data from the YouTube Data API, engineers engagement features  
(view velocity, like-to-comment ratio, trend duration), and trains a classifier to  
predict viral vs. non-viral tracks.

`Python` `YouTube Data API` `Pandas` `Scikit-learn` `Feature Engineering`  
→ [View repository](https://github.com/bhattacharjeeshayan860-netizen/MUSIC-VIRALITY)

### RAG Python Teaching Assistant
*Can a retrieval-augmented LLM actually teach coding — not just autocomplete it?*

Built a RAG pipeline over Python and Data Science documentation. Users ask questions;  
the system retrieves relevant context from a vector store and generates precise,  
grounded answers — no hallucination, no generic responses.

`Python` `RAG` `LangChain` `FAISS` `OpenAI API`  
→ [View repository](https://github.com/bhattacharjeeshayan860-netizen/rag-python-teaching-assistant)
